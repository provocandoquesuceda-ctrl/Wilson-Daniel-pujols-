## 📖 Cómo Usar el Portafolio

### Página Principal (index.html)

#### 1. Navegación
- Usa el menú superior para navegar
- En móvil, haz clic en ☰ (menú hamburguesa)
- Los enlaces tienen scroll suave

#### 2. Contacto
- **WhatsApp:** Abre chat directo
- **Email:** Envía correo electrónico
- **Redes:** LinkedIn y Facebook

#### 3. Secciones
- **Inicio:** Presentación profesional
- **Servicios:** Áreas de práctica
- **Experiencia:** Historial profesional
- **Contacto:** Formas de comunicación

### Aplicación de Tareas (todo.html)

#### Agregar Tareas
1. Escribe en el campo de entrada
2. Haz clic "Agregar" o presiona Enter
3. ¡La tarea aparece en la lista!

#### Completar Tareas
1. Marca el checkbox
2. Se muestra tachada
3. Se cuenta en estadísticas

#### Eliminar Tareas
1. Haz clic "Eliminar"
2. Confirma
3. Se elimina de la lista

#### Limpiar Completadas
1. Usa "Limpiar Completadas"
2. Confirma la acción
3. Se eliminan todas las completadas

#### Estadísticas
- **Total:** Todas las tareas
- **Completadas:** Tareas hechas
- **Pendientes:** Tareas sin hacer

### Almacenamiento Local

✅ Tareas se guardan automáticamente
✅ Se guardan en TU NAVEGADOR
✅ Cierra sin perder datos
⚠️ Limpiar cache = pierdes tareas
⚠️ Hacer backup si necesitas respaldo

## ⚠️ Problemas

### Tareas no aparecen
- Verifica LocalStorage habilitado
- Limpia cache selectivamente

- Limpia cache
- Recarga (F5)
- Verifica conexión

### Menú no funciona
- Actualiza navegador
- Limpia cookies
- Prueba otro navegador
